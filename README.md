# 📝 تطبيق إدارة المهام الذكي

تطبيق بسيط لإدارة المهام باستخدام JavaScript مع إمكانية حفظ البيانات داخل المتصفح باستخدام LocalStorage.

---

## 🚀 مميزات المشروع

- إضافة مهام جديدة
- حذف المهام
- تحديد المهمة كمكتملة
- حفظ المهام داخل المتصفح
- استرجاع المهام بعد إعادة تحميل الصفحة
- واجهة بسيطة وسهلة الاستخدام

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- LocalStorage API

## 🎯 ما تم تطبيقه في المشروع

- التعامل مع DOM
- إنشاء العناصر ديناميكيًا
- التعامل مع الأحداث (Events)
- استخدام المصفوفات (Arrays)
- استخدام LocalStorage
- استخدام JSON.stringify()
- استخدام JSON.parse()
- إضافة وحذف العناصر
- حفظ البيانات واسترجاعها

---

## ▶️ طريقة التشغيل

1. قم بتحميل ملفات المشروع.
2. افتح ملف `index.html`.
3. أضف المهام التي تريدها.
4. جرّب حذف أو إكمال المهام.
5. أعد تحميل الصفحة وستبقى المهام محفوظة.
